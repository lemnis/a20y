A child window within a webpage

* Prompts the user to perform a specific action * If it is designed to interrup, it is usually a modal. See [alertdialog]() * It should have a label, it can be done with the `aria-label` attribute * It should have at least one focusable descendant element. * It should focus an element in the modal when displayed. * It should manage focus of modal dialogs (keep the focus inside the open modal). ##### example
